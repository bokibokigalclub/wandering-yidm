Eirun Last Code ~自架空世界至战场~

Posted at 2018-04-02 MF文库J  MF文库J 东龙乃助 みことあけみ、汐山このむ 

「我真正想守护的
才不是这种虚伪的束缚！」
这是将虚幻化为真实、将幻想化为现实，
不，而是将幻想转化为现实之人灵魂的故事。
西元2070年，人类遭受谜样生命体马里斯的袭击已经度过半世纪以上的时光。
但是唯一能对抗马里斯的巨大兵器〈漆黑魔女〉狄丝特布伦，
其驾驶员赛莲却因充满绝望的战斗精疲力竭，
在这般时日中某天战场却降下奇迹。
国籍不明的谜样战斗机突然破空而出，
用不可能存在于这世界的超级兵器歼灭马里斯。
人类为此欢欣鼓舞然而自战斗机中现身的，
却是与这个世界高人气动画的驾驶员艾伦=巴札特极为神似的少年。
当动画人物转变为现实人物，也引爆一场未知的阴谋——
